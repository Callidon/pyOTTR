Module contents¶

Lexer and parser utilities for manipulating OTTR template definitions and instances.

ottr.parsers.parse_templates(text, format='stottr')¶

Parse a set of stOTTR template definitions. Use the format parameter to set the language used to decalred templates (default to pOTTR). Formats currently supported: stOTTR

ottr.parsers.parse_instances(text, format='stottr')¶

Parse a set of stOTTR template instances. Use the format parameter to set the language used to decalred templates (default to pOTTR). Formats currently supported: stOTTR
